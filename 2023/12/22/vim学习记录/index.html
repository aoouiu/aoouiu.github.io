vim学习记录

本文最后更新于：1 个月前

在命令行输入：vimtutor 进入 VIM 教程

1.1 移动光标

上下左右：k j h l

上上下下 BABA MAYDAY ……

1.2 VIM 进入和退出

按下 ESC 键进入 Normal 模式

wq、q! ......

1.3 文本编辑之删除

normal 模式下 按下 x 键删除光标处的字符

1.4 文本编辑之插入

normal 模式下 按下 i 键在光标当前处插入字符

1.5 文本编辑之添加

normal 模式下 按下 A 键在当前行的末尾插入字符

2.1 删除类命令

normal 模式下 dw 删除光标处至下一个单词首字母（不包含下一单词的首字母）

2.2 更多删除命令

normal 模式下 d$ 删除光标至行末

2.3 关于命令和对象

normal 模式下 d motion

d - 删除操作符
motion - 操作对象

motion :

  • w：当前光标所在单词的下一个单词的首字符(但不包含)
  • e：当前光标所在单词的末尾
  • $：当前光标所在行的行末

dw、de、d$

2.4 使用计数指定动作

normal 模式下

2.5 使用计数以删除更多

normal 模式下 d number(数字) motion

2.6 操作整行

normal 模式下 dd 删除光标所在行

2.7 撤销类命令

  • u —————- 撤销修改 backward
  • CTRL + r 键 —— 恢复撤销的修改 forward
  • U —————— 撤销整行修改

3.1 置入类命令

normal 模式下

  1. 当使用 dd 删除行后，该行的内容被保存到 vim 的一个寄存器中，使用 p 键粘贴被删除行内容
  2. 在光标所在行使用 p 键，会将内容粘贴到当前行的下一行

3.2 置换类命令

normal 模式下

输入 r 和一个字符，替换掉光标所在处的字符

3.3 更改类命令

normal 模式下

ce 从当前光标至当前光标所在单词的尾字符（且包含尾字符），将这些字符删除并进入 insert 模式

3.4 使用c更改更多

normal 模式下 c [number] motion

例如：c2w c3e

4.1 定位及文件状态

1，按下 CTRL 键 + g 键，显示当前行号等信息

2，按 G 键，跳转到文件最后一行；gg 键跳转文件第一行

3，在 normal 模式下，直接输入行号再按 G 键，跳转到该行

4.2 搜索类命令

1，在 normal 模式下，直接输入 / + 关键字回车，即可查找关键字

2，n 下一个匹配项，N 上一个匹配项

4.3 配对括号的查找

1，移动光标到 ( [ { 处，接着按下 % 键

2，跳转到匹配的括号处

3，再按下 % 键，跳回

4.4 替换命令

1，:s/thee/the + 回车：只会匹配当前行中的 “thee” 并且替换为 “the”，如果当前行没有thee则查找失败

2，:s/thee/the/g + 回车：会将该行中所有的 “thee” 替换为 “the”
